✨ Features

ฟีเจอร์ของ วิล เซอร์ฟอท (Bot)

ครบทุกฟีเจอร์ที่เซิร์ฟเวอร์ต้องการ ในบอทเดียว 💙

🔊

ออนช่อง Voice อัตโนมัติ

บอทออนช่อง Voice ตลอด 24 ชม. ถ้าหลุดออกจากช่อง ระบบจะกลับเข้าใหม่อัตโนมัติทันที

✏️

เปลี่ยนชื่อช่องตามเวลา

ตั้งตารางให้บอทเปลี่ยนชื่อห้อง Voice อัตโนมัติ เช่น เปิด/ปิด ตามช่วงเวลาที่กำหนด

📊

Dashboard แบบ Embed

ควบคุมบอทผ่านปุ่มใน Discord ได้เลย ไม่ต้องพิมพ์คำสั่ง ดูสถานะและตั้งค่าได้ในที่เดียว

🚫

ระบบ Blacklist

แบนคน ยศ หรือเซิร์ฟเวอร์ไม่ให้ใช้บอทได้ มีระบบป้องกันการใช้งานในทางที่ผิด

📨

ส่ง DM อัจฉริยะ

ส่งข้อความ DM หาคนเดียว หรือ DM ทุกคนในเซิร์ฟพร้อมกัน เฉพาะเจ้าของเซิร์ฟเท่านั้น

💾

ฐานข้อมูล SQLite

ตั้งค่าทุกอย่างถูกบันทึกลง Database รีสตาร์ทบอทข้อมูลไม่หาย โหลดค่าคืนอัตโนมัติ

🎮 Commands

Slash Commands

คำสั่งทั้งหมดที่ใช้งานได้บน Discord 💙

⚙️
/setup
เปิดแผงควบคุม Dashboard ตั้งค่า VC, ตาราง และ Blacklist ได้ครบในที่เดียว
ทุกคน
💬

ทุกคน
📩
/dm send
ส่ง DM ส่วนตัวไปหาสมาชิกคนใดก็ได้ผ่าน User ID พร้อมระบุข้อความที่ต้องการ

📢
